# Object Oriented Shapes (C++)

This project is a C++ tool designed to calculte the **area** and  **perimeter** of Geometric Shapes


## Features
* **Inheritance:** Uses a base class for generic shapes.
* **Polymorphism:** Implements virtual functions for dynamic area calculations.
* **Clean Code:** Structured with headers and implementation files.

##  How to use 
1. Clone the repository .
2. Compile the 'main.cpp' using a C++ Compiler (e.g., g++).
3. Run the exwcurtable to see the calculations for different shapes.
